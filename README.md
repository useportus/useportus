## About Portus

Portus builds focused WooCommerce tools for real stores.

Our products are designed to improve the shopping experience, simplify store operations, and help merchants sell with less friction.

## First Product

### Portus Cart for WooCommerce

A fast side cart for WooCommerce with AJAX quantity updates, item removal, a floating cart button, theme compatibility settings, and a clean mobile-first experience.

- Built for WooCommerce
- Designed for mobile commerce
- Compatible with common WordPress themes
- Developed with a Free and Pro roadmap
